# WIN-HOUSE Brand Assets

文浩集團對外公開品牌資產儲存庫。本 Repo 透過 jsDelivr CDN 全球加速,可供 AI 工具、簡報、文件、網站直接引用。

---

## 📦 資產目錄

| 路徑 | 內容 | 狀態 |
|---|---|---|
| `/wingo/` | WINGO 吉祥物姿態與表情 | Active |
| `/logos/` | Win-House 集團 Logo(預留) | Planned |
| `/design-system/` | Design System tokens.json(預留) | Planned |

## 🎨 品牌色票對照

| Token | HEX | 用途 |
|---|---|---|
| Gold(主色) | `#C9941F` | 強調、正向、達標、Do 準則 |
| Logo Red | `#E1251B` | 僅限 Logo 使用 |
| Slate Blue | `#4A6B8A` | 調和色、資訊、負向、Don't 準則 |
| Clay | `#B57258` | 錯誤、警示 |
| Sage | `#5A8159` | 成功狀態 |

## ⚖️ 使用規範

### ✅ 適用場景
- 內部工具(職級地圖、KPI Dashboard)
- LINE 自動週報
- 內部訓練教材與簡報
- 系統 Empty State 與成功提示

### ❌ 排除場景
- 官網 Hero 與產品型錄
- Datasheet 與原廠技術文件
- 報價單、契約、發票
- Tier 1 客戶提案封面
- 原廠 Co-branded 行銷素材
